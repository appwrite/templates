# ⚡ Rust Starter Function

A simple starter function. Edit `main.rs` to get started and create something awesome! 🚀

## 🧰 Usage

### GET /ping

- Returns a "Pong" message.

**Response**

Sample `200` Response:

```text
Pong
```

### GET, POST, PUT, PATCH, DELETE /

- Returns a "Learn More" JSON response.

**Response**

Sample `200` Response:

```json
{
  "motto": "Build like a team of hundreds_",
  "learn": "https://appwrite.io/docs",
  "connect": "https://appwrite.io/discord",
  "getInspired": "https://builtwith.appwrite.io"
}
```

## ⚙️ Configuration

| Setting           | Value       |
| ----------------- | ----------- |
| Runtime           | Rust (1.83) |
| Entrypoint        | `main.rs`   |
| Permissions       | `any`       |
| Timeout (Seconds) | 15          |

## 🔒 Environment Variables

No environment variables required.
